# lims is a Library Information Management System Project 
# Developed by Aron Zeru
This Library Information Management System runs on Django Framework in the back-end and HTML, and CSS in the front-end.

**Futures:**
This is a Library Information Management System which contains the following futures
=>Home page that contains general ingormation about the system. 
=>Readers page – In this feature, the admin can manage all the information of the readers including, “add new reader”, “update or modify reader” and “delete reader “.
=>Books page – In this feature, the admin can manage all the information of books including, “add new book”, “update or modify book” "Issue a book to reader" and “delete book “.
=>Issued page – In this feature, we will be able to see the list od issued books along with name and ID of the readers who borrowed the books.
=>Login/Logout System – With this feature, the user can log in and log out of the system.

**Steps to run the project:**
Step one: download and unzip the LibraryInformationManagementSyatem folder.
Step two: Open your VScode editor and navigate to that folder.
Step three: type "PythonEnv\Scripts\activate" and press Enter key on the keyboard #This will activate the virtual environment.
Step Four: type "cd lims_portal" and press Enter key on the keyboard # This will change the directory in to the app folder.
Step Five: type "python manage.py runserver 8080" press Enterkey on the keyboard # This will make the project to run on port 8080.
